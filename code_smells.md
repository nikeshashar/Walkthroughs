Code Smell --> Refactoring
===================

* long method  --> extract method
* obscure var/method names --> rename
* repetition (duplicated code) --> extract commonality
* nested ifs --> extract method
* magic numbers (extract constants) --> extract constant
* responsibilities
* indentation
